Operadores Matematicos

  • Soma -> +
  • Subtração -> -
  • Multiplição -> *
  • Divisão -> /
  • Resto da divisão -> %
  • Increment -> ++
  • decrement -> --
  • raiz -> Math.sqrt(4,2)
  • potencia -> Math.pow(4,2)

Operadores Logicos

  • AND -> &&
  • OR -> ||
  • NOT -> !

Operadores de comparação

  • Igual a o valor -> ==
  • Igual a o valor e o tipo -> ===
  • Diferente ao valor-> !=
  • Diferente ao valor e o tipo-> !==
  • Maior que -> >
  • Menor que -> <
  • Maior e igual -> >=
  • Menor e igual -> <=

Lacos de repetição

  • For
    for (var x=0;x<10; x++){ }
  • While
    while( i < 10){ i++ }
  • do while
    do{ i++ }while( i > 10)
