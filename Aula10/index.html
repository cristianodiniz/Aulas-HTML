CSS Layout - Propriedade position

A propriedade position o metodo de posicionamento que o elemneto terá.

A propriedade pode ter os seguintes valores

  • static
  • relative
  • fixed
  • absolute
  • sticky

Para posicionar o elemento usa-se as propriedades a seguir

  • top
  • bottom
  • left
  • right

Essas propriedade somente funciona quando usado combinado a propriedade position respondida diferente de static

position: static

A reposta defaul dos elementos html, e quando statico nao tem efeito top, botton e left e right. Nesse caso os elementos são posicionados conforme a fluxo da pagina normal.

  • Exemplo 1 - position: static

position: relative;

Um elemento com posição: relative; está posicionado com relacao a sua posicao normal.

  • Exemplo 2 - position: relative

position: fixed;

Um elemento com posição: fixed; está posicionado com relacao a viewport, isso significa que ele sempre ficará visivel em algum lugar da tela, mesmo que a pagina seja rolada.

  • Exemplo 3 - position: fixed

position: absolute;

Um elemento com posição: absolute; está posicionado com relacao ao elemento pai mais proximo, podendo ser ajustado usando as propriedades top, botton e left e right.

  • Exemplo 4 - position: absolute

position: sticky;

Um elemento com posição: sticky; é posicionado com base na posição de rolagem do usuário.

  • Exemplo 5 - position: sticky

Sobreposição de Elementos z-index

Quando o elemento esta posicionado, o mesmo pode sobrepor outros elementos

Para contolar este caso usa-se a propriedade z-index para especificar a ordem da pilha de vizualização. Ou seja se o elemento ficara sobre, atras ou no mesmo contexto que os demais.

  • Exemplo 6 - z-index
  • Exemplo 7 - z-index

Nota: Se dois elementos posicionados se sobrepuserem sem um índice z especificado, o elemento posicionado por último no código HTML será mostrado na parte superior.
