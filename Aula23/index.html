Configuracao

Para fazer o testes execute os passos a seguir

  • Instalar o node js - Passo-a-passo
  • Instalar o http-server executando o comando no terminal
  • sudo npm install http-server -g

    • Caso de problema de versao execute os comandos

  • sudo npm cache clean -f
  • sudo npm install -g n
  • sudo n stable

    • Agora é so executar o comando http-server na pasta que deseja colocar online

  • Libera a porta 8080
  • sudo iptables -I INPUT -s 0/0 -d 0/0 -p tcp --dport 8080 -j ACCEPT
