Cores

As cores podem ser especificadas por:

  • Nome - Tomate
  • RGB - Composicao das cores primarias RED GREEN e BLUE - rgb(255,0,0)
  • HEX - Composicao das cores primarias RED GREEN e BLUE so que em hexadecimal #00FF00
  • RGBA - RGB com alpha rgba(255, 0, 0, 0.5)
  • HSL - Compoiscao é definida pela tonalidade saturacao e brilho hsl(60, 10%, 50%)
  • HLSA - HSL com a adicao do alpha hsla(180,100%,50%, 0.5)
