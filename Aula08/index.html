CSS Layout - float and clears

A propriedade Float do CSS especifica como um elemento deve flutuar.

  • Exemplo 1 - Direita e Esquerda

A propriedade float pode ter um dos seguintes valores

  • left - O elemento flutuará para a esquerda do seu elemento container;
  • right - O elemento flutuará para a direita do seu elemento container;
  • none - O elemento nao flutuará, será renderizado conforme o contexto do mesmo. Este é o valor default;
  • inherit - O elemento inherit flutuará conforme o elemento pai.

Em sua aplicação simples a propriedade float pode ser usada para quebra o texto em torno de uma imagem.

The clear Property

A propriedade Clean do CSS especifica como um elemento não deve flutuar.

A propriedade clean pode ter um dos seguintes valores

  • none - Permite flutuar para os dois lados. Este é default
  • left - Nao permite flutuar para a esquerda
  • right- Nao permite flutuar para a direita
  • both - Nao permite flutuar para a para os dois lados
  • inherit - Herda do elemento pai o atributo clear

O uso mais comum dessa propriedade é depois de usar o float em um elemento.

Quanto devo usar o clear? Pode se usar o clear para limpar o float, como por exemplo, quando existe um elemento flutuando a esquerda, e logo abaixo existe outro elemento. Sem o clear o segundo elemento sera renderizado na mesma posição do elemento que esta flutuando. Sendo assim use o clear:left para o sengundo elemento, para que o mesmo seja renderizado abaixo do primeiro.

The Clearfix Hack

Quando o elemento pai é menor que o elemento filho, sendo que o filho tem a propriedade float setada, veremos que o elemento filho ficará fora do elemnto pai.

Para isso podemos usar a propriedade overflow:auto para corrigir esse problema, essa abordagem é chamada de "Clearfix Hack".

  • Exemplo 7 - sem clearfix
  • Exemplo 8 - com clearfix

Cabe comentar que o overflow:auto nao resolve todos os casos, sendo que a abordagem melhor para isso é usar pseudo-element.
