CSS Layout - Overflow

A propriedade Overflow controla oque acontece com o conteudo quando ele é maior que o container.

A propriedade Overflow pode ser respondida com:

  • visible - Default. Não restrige o conteudo dentro do container, deixando vazar;
  • hidden - Restrige o conteudo dentro do container, não deixando vazar;
  • scroll - Restrige o conteudo dentro do container, não deixando vazar e adiciona uma scroll para conseguir ver o resto do conteudo;
  • auto - Restrige o conteudo dentro do container, não deixando vazar e adiciona uma scroll para se o conteudo for maior que o container.

Obs: O overflow somente funciona em elementos block com height informado.

Overflow-x e Overflow-y

O overflow-x and overflow-y especifica onde vai ser controlado o conteudo se é horizontal ou vertical.

  • overflow-x - horizontal
  • overflow-y - vertical
