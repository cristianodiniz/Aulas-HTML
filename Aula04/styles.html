HTML - Styles

para adicionar um estilo em um elemento HTML, pode ser feito usando ao atributo style.
Seguindo a sintaxe abaixo:

<tagname style="property:value;">

Algumas propriedades do Style

  • Usa-se background-color definir a cor de fundo
  • Usa-se color definir a cor
  • Usa-se font-family a familia da fonte
  • Usa-se font-size para definir o tamanho da fonte
    • width
  • Usa-se text-align para alinhar texto
