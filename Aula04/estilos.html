Propriedades do style

  • Usa-se color para definir a cor do texto.
  • Usa-se background-color para definir a cor do fundo.
  • Usa-se font-family para definir a fonte do texto.
  • Usa-se font-size para definir o tamanho do texto.
  • Usa-se font-align para definir o alinhamento do texto.
  • Usa-se border para definir a borda do elemento.
